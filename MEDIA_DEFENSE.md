# L'Anthropie — Défense face aux interlocuteurs hostiles ou pressés

> *Quatre outils sobres pour répondre sans se laisser entraîner : un questionnaire à imposer en amont, trois scripts pré-rédigés en 4 blocs, un compte rendu à froid après chaque échange tendu, et une préparation 7 minutes en salle d'attente. Pour journalistes à charge, chercheurs hostiles, parents inquiets, comités d'éthique sceptiques, ou porteur isolé entrant dans un rendez-vous administratif.*

## Outil 1 — Questionnaire de la Partie Adverse (avant toute réponse)

Quand un·e journaliste, chercheur·euse hostile, comité d'éthique, autorité administrative vous contacte avec une demande implicite ou explicite de défense, **ne jamais répondre tout de suite**. Imposer d'abord ce mini-questionnaire.

### Le questionnaire (à envoyer par mail)

```
Bonjour [Prénom NOM],

Merci pour votre message. Avant de répondre sur le fond, je vous propose
de fixer cinq précisions qui aideront un échange utile à tous les deux.

1. ANGLE
   Quelle est la thèse ou la question centrale de votre travail
   sur le sujet ? Une formulation en une phrase suffit.

2. CITATIONS
   Si vous comptez me citer, sous quelle forme : citation directe
   par mail relu, paraphrase, mention sans citation ? Si citation
   directe, je demande validation explicite avant publication.

3. MÉTHODE
   Comment recueillez-vous les voix opposées à la vôtre ?
   Avez-vous interrogé d'autres voix critiques de l'édifice
   Anthropie ? Lesquelles ?

4. DÉLAI
   Quelle est votre échéance ? Je m'engage à répondre sous 7 jours
   ouvrés à partir de votre retour sur les 4 points ci-dessus,
   pas avant.

5. DROIT DE CORRECTION
   Si après publication je relève une erreur factuelle (citation
   erronée, fait inventé, source mal attribuée), engagez-vous
   à publier une correction visible dans le délai usuel de votre
   média / institution ?

Une fois ces cinq points reçus par écrit, je réponds au fond.

Cordialement,
[Vous]
```

### Effet

- **Déplace l'échange du duel vers le protocole.** Les acteurs hostiles comptent souvent sur la pression du délai pour obtenir des phrases mal pesées.
- **Filtre les démarches sérieuses des démarches biaisées.** Un·e journaliste sérieux·euse répondra aux 5 points dans 24-48h. Un·e journaliste à charge ou paresseux·euse se retirera.
- **Vous donne une trace écrite** opposable si la publication finale détourne vos propos.
- **Réduit l'asymétrie d'information**. Vous n'êtes plus en réaction, vous êtes en cadrage commun.

### Quand ne pas l'utiliser

- Pour un échange entre porteurs Anthropie sur Issues GitHub publiques (le cadre est déjà transparent).
- Pour un échange interne à votre prototype (cellule, atelier, etc.).
- Pour une demande administrative routinière (école, mairie, services publics dans leur fonction normale).

## Outil 2 — Réponse 90 secondes par profil (3 scripts pré-rédigés)

Trois scripts à mémoriser ou imprimer A6, à utiliser quand vous **devez** répondre en moins de 2 minutes (entretien filmé, échange de couloir, question de fin de présentation publique). Format 4 blocs : *fait observable / limite admise / risque reconnu / renvoi vérifiable*.

### Profil A — Chercheur·euse hostile

> *« Vous demandez si l'édifice Anthropie a une validation empirique. **Le fait observable** est qu'il n'en a pas — c'est explicitement reconnu dans `EVIDENCE_MAP.md` qui classe les claims de niveau A à D. **La limite admise** est que les cinq inventions propres (PAV, Liminalité Protégée, Anti-Pathologisation, etc.) sont au niveau D — non validées longitudinalement. **Le risque reconnu** est qu'elles peuvent ne jamais l'être si aucune cohorte ne les traverse, et l'édifice prévoit l'arrêt acceptable dans REFUTATION.md §6. **Le renvoi vérifiable** : tout est documenté dans `anthropie.org/lire/preuves/`. Critique radicale bienvenue. »*

### Profil B — Parent inquiet

> *« Vous vous demandez si Anthropie peut nuire à un·e enfant. **Le fait observable** est que la doctrine `SAFETY.md` pose explicitement protection mineurs, non-thérapie, anti-secte, multi-référents, casier vierge, transparence parentale absolue. **La limite admise** est qu'aucune doctrine n'empêche un usage abusif par quelqu'un de mauvaise foi qui se réclamerait de l'édifice. **Le risque reconnu** est que l'édifice CC0 anonyme n'a pas de pouvoir d'exclusion juridique — seulement de désolidarisation publique. **Le renvoi vérifiable** : si vous identifiez un usage abusif, signalement Issues GitHub tag safety + autorités civiles compétentes (Miviludes, CIC, équivalent national). »*

### Profil C — Journaliste à charge

> *« Vous suggérez qu'Anthropie est lié à Anthropic ou à un labo IA. **Le fait observable** est qu'aucun lien institutionnel, organisationnel ou financier n'existe — le fichier `NOT_AN_AI_LAB.md` et `NAMING.md` documentent explicitement la séparation. **La limite admise** est que le nom phonétique-ment proche peut induire en erreur, c'est pourquoi nous avons publié ces clarifications. **Le risque reconnu** est qu'une couverture imprécise transmette la confusion à un public qui ne lira pas la précision. **Le renvoi vérifiable** : `anthropie.org/lire/pas-un-labo-ia/` et `anthropie.org/lire/nommage/`, tous deux datés et publics. Citez-moi si vous voulez ; je vous demanderai relecture avant publication. »*

### Pourquoi 4 blocs et pas plus

- *Fait observable* protège contre la tentation d'argumenter en abstrait.
- *Limite admise* signale que vous n'êtes pas dans le déni.
- *Risque reconnu* coupe l'attaque par projection (*« mais si... »*).
- *Renvoi vérifiable* rend l'échange traçable et vérifiable indépendamment de votre parole.

### Pour adapter à votre profil

Vous pouvez préparer **votre propre script en 4 blocs** sur un point de l'édifice qui vous est régulièrement opposé. Conservez la structure ; adaptez les références.

## Outil 3 — Compte rendu à froid (après chaque échange tendu)

Dans les 24h qui suivent un échange hostile (entretien, débat, débrief institutionnel), produire une **note de 7 lignes** dans votre carnet personnel — pas pour publication, pour vous.

### Format de la note

```
Date : __________
Interlocuteur·trice (rôle, pas nom obligatoire) : ______________________
Contexte : ______________________________________________________

1. Une question qui m'a été posée que je ne sais pas trancher :
   _______________________________________________________________

2. Une réponse que j'ai donnée que je regrette ou doute :
   _______________________________________________________________

3. Un point non tranché qui mériterait suite :
   _______________________________________________________________

4. Ce qui m'a énervé, et ce que ça révèle sur moi (pas sur l'autre) :
   _______________________________________________________________

5. Décision prise à froid : ouvrir Issue / amender un fichier /
   accepter le désaccord / ne rien faire / autre : ________________

6. Date de relecture (1 mois) : ________________
```

### Effet

- **Réduit la déformation rétrospective**. À chaud, on se souvient mal de l'échange.
- **Coupe la mémoire militante**. Sans la note, on rumine la victoire ou la défaite ; avec la note, on rumine l'apprentissage.
- **Honore le désaccord** sans devoir le résoudre tout de suite.
- **Articulé avec la doctrine de l'édifice** : un compte rendu à froid peut alimenter [`OPEN_GRIEVANCES.md`](griefs/) si une critique sérieuse mérite figurer publiquement.

### Quand relire à 1 mois

À la date prévue, relire la note. Demander :

- Est-ce que je vois différemment maintenant ?
- Est-ce que la décision prise à froid a été appliquée ? Si non, pourquoi ?
- Est-ce qu'un échange similaire s'est répété — pattern à documenter ?

## Outil 4 — Préparation 7 minutes en salle d'attente

> *Fiche ultra-courte pour parent isolé, prof seul, soignant burn-out, ou tout porteur qui s'apprête à entrer dans un rendez-vous hostile ou indifférent. À lire dans la salle d'attente, pas avant. Idéalement imprimée recto-verso A4, pliée en deux, glissée dans une poche intérieure ou un sac.*

Vous l'ouvrez parce que vous allez entrer dans un bureau d'école, de mairie, d'hôpital, de service social, de commission, de tribunal — un rendez-vous où vous savez que la personne en face risque d'être hostile, méprisante, indifférente, fatiguée, ou en position de pouvoir excessive. Vous avez **7 minutes** pour vous préparer. Cette fiche ne vous dit pas ce qu'il faut dire — elle vous aide à savoir **où vous mettre** pour pouvoir parler.

### 1 minute — Objectif unique

À voix basse ou dans votre tête : *« Qu'est-ce que je veux **obtenir** dans les 60 prochaines minutes ?* » Pas plusieurs choses. **Une**. Choisissez la plus importante : un rendez-vous écrit avec date, le refus tracé d'un protocole médicamenteux, la reconnaissance par écrit qu'un fait s'est produit, un délai administratif, un compte rendu mail.

Si vous obtenez cet objectif, le rendez-vous est réussi. Tout le reste est bonus. Si l'autre personne vous embarque sur 5 sujets, ramenez à votre objectif unique. Politiquement, calmement.

### 2 minutes — Trois pièces

- **Pièce A — Faits vérifiables** : 1 à 3 phrases qui se vérifient indépendamment de votre parole (date, document, témoin tiers, certificat).
  > *« Le 12 mars, mon enfant a manqué l'école pour [...]. Voici le certificat médical du Dr X. »*
- **Pièce B — Demande exacte** : ce que vous demandez, en une phrase opposable (qu'on puisse vous accorder ou vous refuser sans ambiguïté).
  > *« Je demande un aménagement scolaire selon l'article L131-2. »*
- **Pièce C — Limite non-négociable** : ce que vous n'accepterez pas, quoi qu'il arrive. Calmement. Sans menace.
  > *« Je ne signerai aucun document aujourd'hui. »*

### 1 minute — Phrase de recadrage

Une seule phrase, à mémoriser, prononcée calmement quand l'autre dévie ou attaque :

> *« Je vous remercie pour cette information. Pour aujourd'hui, ma demande reste celle-ci [...]. »*
> *« Je note ce que vous me dites. Je reviendrai vers vous par écrit après réflexion. »*
> *« Pouvez-vous me confirmer cela par écrit, s'il vous plaît ? »*

Cette phrase n'est pas une victoire. Elle est une **ancre**. Elle vous ramène.

### 1 minute — Phrase de sortie

Si l'entretien dérape ou vous met en difficulté psychologique majeure, vous **partez**. Sans culpabilité.

> *« Je vous remercie. Cet échange ne peut plus continuer aujourd'hui dans des conditions productives. Je reviendrai vers vous par écrit. »*

Vous vous levez, vous saluez, vous sortez. Personne ne peut vous obliger à rester dans un rendez-vous administratif (sauf rares cas pénaux ou hospitaliers contraints — et ceux-là, vous le savez).

### 1 minute — L'après-coup

Quand vous sortez — bon ou mauvais —, trois choses dans cet ordre :

1. Endroit calme (banc, café, voiture). Trois respirations lentes.
2. **Notez par écrit** : qui, quand, où, ce qui a été dit, refusé, accordé, flou.
3. **Mail récapitulatif** dans les 24h : *« Suite à notre rendez-vous du [date], je note les points suivants : [...]. Pourriez-vous me confirmer ces points ? »*

Cette trace écrite **est** votre seconde voix. Elle existe quand la mémoire orale s'efface ou est contestée.

### 30 secondes — Si vous tremblez

Avant de pousser la porte : **trois respirations** lentes (4 sec inspiration, 6 sec expiration), pieds posés au sol, une main sur la cuisse ou le ventre, redressez-vous sans raideur. Vous n'avez pas à *paraître* confiant·e. Vous avez à *rester* vous-même. C'est différent et plus solide.

### 15 secondes — Sortir de la fiche

Si vous avez 3 minutes de plus dans la salle d'attente, **rangez la fiche**. Vous l'avez assez lue. Continuer à la regarder vous épuise. Les trois pièces — *Faits / Demande / Limite* — suffisent.

### Si la situation dépasse votre capacité

L'Outil 4 n'est pas suffisant si :

- Détresse psychique aiguë (idéation suicidaire, dissociation, panique majeure) → professionnel **avant** le rendez-vous. **143** (Suisse), **3114** (France), [findahelpline.com](https://findahelpline.com).
- Procédure pénale ou placement judiciaire → vous avez droit à un avocat. Demandez-le.
- Violences (conjugales, familiales, institutionnelles aggravées) → **117** (Suisse), **17** ou **3919** (France).

L'édifice ne se substitue à aucune protection légale ni soin médical. Cf. [`SAFETY.md` §3](garde-fous/).

> *« Sept minutes ne suffisent pas pour gagner. Sept minutes peuvent suffire à ne pas se perdre. »*

## Pour les premiers porteurs publics

Si vous êtes la première personne à parler publiquement de l'édifice Anthropie sous votre nom (article, interview, conférence) :

1. **Lire ces trois outils avant.**
2. **Préparer votre script 4 blocs** sur le point que vous voulez articuler.
3. **Demander le questionnaire en amont** à tout·e journaliste / chercheur·euse.
4. **Tenir le compte rendu à froid** après chaque échange.
5. **Si vous craquez** (réponse mal donnée, citation hors contexte) : ouvrir Issue GitHub avec tag `safety` ou `lecture-critique`, documenter votre erreur, proposer correction. Pas de honte. C'est doctrinal.

## Limites de ces outils

- **Ne remplacent pas un travail de communication structuré** si vous portez l'édifice à grande échelle. À ce moment, médiateur·trice juridique (cf. [`MANDATES.md`](mandats/)) avec compétences communication devient pertinent.
- **Pas une protection contre les attaques juridiques.** Si un État ou une entreprise déclenche une procédure, consulter un·e avocat·e compétent·e dans la juridiction concernée. Cf. [`LEGAL_BOUNDARIES.md`](cadre/).
- **Pas une excuse pour parler en mauvaise foi.** Le questionnaire et les scripts servent à clarifier l'échange, pas à éviter de répondre sur le fond.

## Articulation avec l'édifice

- **Couche 4 — LA PAROLE** : ces outils opérationnalisent la *Discipline de Prédiction-Avant-Vocal* dans un contexte hostile spécifique. Avant de répondre, prédire ce qui sera entendu, ce qu'un LLM dirait, ce qui est attendu, et choisir conscient.
- **`HOSTILE_DRILL.md`** : ces outils sont pour les attaques **réelles**. HOSTILE_DRILL est pour les attaques **simulées**. Les deux se complètent.
- **`OBJECTIONS.md`** : si une attaque s'appuie sur une objection sérieuse non encore traitée, l'ajouter à OPEN_GRIEVANCES.md.
- **`SAFETY.md`** : si une attaque révèle un trou dans les garde-fous éthiques, amender SAFETY.md par PR avec discussion publique.
