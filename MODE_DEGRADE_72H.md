# L'Anthropie — Mode Dégradé 72H (carte A6 pour porteur fatigué)

> *Une carte A6 imprimable, activable pendant 72 heures, qu'un porteur fatigué affiche dans son contexte (bureau, salle de classe, atelier, foyer) **sans se justifier**. Transforme la fatigue silencieuse en protocole lisible.*

CC0 1.0 Universal. Cf. codex iter#24 #1. Outil minimaliste pour les moments où l'on doit ralentir sans avoir à expliquer pourquoi.

**À utiliser** : quand vous animez un dispositif Anthropie (cellule mycéliale, atelier, prototype) et que vous traversez une période de fatigue, deuil, surcharge, ou maladie. **Pas un dispositif pour cacher un effondrement** — si vous traversez une vraie crise, lire [`SAFETY.md` §3](garde-fous/) et orienter vers un·e professionnel·le.

---

## La carte A6

Format imprimable A6 (105 × 148 mm), recto-verso. À photocopier, plier dans la poche, afficher sur la porte ou la table de réunion.

## Comment imprimer / fabriquer

### Option simple

1. Recopiez le recto-verso à la main sur une fiche A6 (ou A5 plié en deux).
2. Datez quand vous l'activez.
3. Affichez à un endroit visible (porte de bureau, table de cuisine, écran d'ordinateur).

### Option imprimable

Imprimer le contenu ci-dessus en A6 sur papier sobre (kraft, blanc cassé). Garder un stock de 5-10 cartes vierges en réserve — si la fatigue est récurrente, vous en aurez besoin.

### Option numérique

Une signature email automatique pendant 72h, avec le contenu du recto :

```
[Mode dégradé activé du JJ/MM au JJ/MM]
Pendant ces 72h : reporter / raccourcir / déléguer / annuler.
Je réponds aux questions vitales uniquement.
Je n'ai pas à me justifier.
```

## Ce que cette carte fait

### Pour vous

- **Protège votre énergie** sans avoir à argumenter à chaque sollicitation.
- **Rend visible** une fatigue normalement invisible (vous n'avez pas à dire *« je suis fatigué·e »* — la carte le dit pour vous).
- **Pose une limite temporelle** : 72 heures, ni plus ni moins. Pas un mode dégradé permanent.
- **Vous oblige à reprendre** au bout de 72h. Si vous ne pouvez pas reprendre, c'est un signal — voir un·e médecin / thérapeute / proche.

### Pour les autres

- **Évite les explications déçues** (*« mais pourquoi tu ne réponds pas »*).
- **Donne une option** : ils peuvent attendre 72h, ou trouver une alternative (déléguer ailleurs, reporter eux-mêmes).
- **Préserve la relation** : ce n'est pas un refus relationnel, c'est une pause technique.

---

## Ce que cette carte NE fait pas

- ❌ **Pas une excuse permanente.** Vous ne pouvez pas activer le mode dégradé toutes les semaines. C'est un dispositif d'exception, pas un mode de vie.
- ❌ **Pas un substitut à un arrêt maladie.** Si vous êtes vraiment malade, voir un·e médecin et arrêter formellement.
- ❌ **Pas un cadre pour ignorer les obligations vitales** (sécurité d'enfants, soin d'une personne dépendante, urgence médicale).
- ❌ **Pas un signal pour fuir un conflit.** Si vous avez un conflit ouvert, le différer 72h peut aggraver — préférer médiation directe.

---

## Quand activer

### Bonnes raisons d'activer

- Burn-out naissant (fatigue qui dépasse une nuit de sommeil).
- Deuil dans les 30 premiers jours.
- Maladie aiguë non hospitalisable.
- Surcharge ponctuelle (déménagement, dossier urgent, examen).
- Crise familiale (séparation, accident, problème enfant).
- Lendemain de gros événement (accouchement, mariage, enterrement).

### Mauvaises raisons d'activer

- ❌ Procrastination chronique (*« je me sens flemmard depuis 2 mois »*).
- ❌ Évitement d'un conflit interpersonnel (*« je ne veux pas répondre à X »*).
- ❌ Raisons stratégiques ("« mode VIP », « si je réponds vite ils penseront que je suis pas occupé»).
- ❌ Manipulation (activation comme pression sur l'entourage).

Si vous hésitez sur la légitimité de votre activation, demandez à **un·e proche de confiance** : *« Penses-tu que c'est raisonnable que j'active mon mode dégradé 72h ? »* Cinq personnes sur cinq vous diront oui si c'est légitime ; trois sur cinq seront perplexes si ça ne l'est pas.

## Pour les communautés et porteurs en charge

Si vous animez une cellule mycéliale, un atelier régulier, un prototype, et que vous activez le mode dégradé :

1. **Annoncez** à votre groupe par message court : *« Mode dégradé 72h activé du JJ au JJ. Animation reportée / raccourcie / déléguée à X. »*
2. **Proposez** un substitut s'il existe (autre porteur, format léger, pause assumée).
3. **Reprenez** après 72h sans drama, sans excuse, sans rattrapage.

Si la cellule mycéliale ne tient pas vos 72h d'absence, c'est une donnée — la cellule était plus dépendante de vous qu'elle ne devrait l'être (cf. [`MANDATES.md`](mandats/) §bus factor minimal).

## Pour modifier cette carte

Pull Request avec votre adaptation. Doctrine demandée :

1. **72h maximum.** Pas 7 jours, pas 30 jours.
2. **Pas de justification due.** Le mode est lisible, pas argumenté.
3. **Garde-fous vitaux préservés.** Sécurité des mineurs / vulnérables / vies en jeu = jamais suspendus.

Variantes proposables : carte 24h pour fatigue ponctuelle, carte 7 jours pour deuil aigu (avec justification médicale), carte familiale (pour parent), carte d'équipe (pour responsable).
